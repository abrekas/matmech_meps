### Мысль:
1. подготовить исходную картинку (а может даже и полигонами подготовить)
2. по подготовленной картинке нарисовать красивый граф (т.е. без ненужных изгибов)
3. вернуть граф (координаты точек по которым строим ломаную)

### Файлы:
1. `graph_builder.py` - код по картинке делает граф (пока только картинки рисует)
2. `preprocessed_plan5.png` - подготовленный план этажа 
3. `corners.py` - находит углы на черновой картинке графа
